UC2M1 Commander Onduleur Cisaille

UMLUseCase
SystemeBord_Aerostat :: ArchitectureLogiciel_SystemeBord :: UC2M1 Commander Onduleur Cisaille

Description

Cas d'utilisation : Commander l'Onduleur de la cisaille lorsque les interrupteurs sont fermés Acteurs : Onduleur (composé de 4 IGBT ou MOFSET) et Interrupteurs Point de vue : Microcontrôleur 1 Pré-conditions : Module de communication actif et fonctionnel, microcontrôleur actif et fonctionnel, onduleur alimenté et en bon état de fonctionnement, interrupteurs alimentés et en bon état de fonctionnement. Déroulement Normal : - Description détaillée : Les deux interrupteurs sont fermés. - Le microcontrôleur envoie les signaux correspondants à chaque IGBT pour faire fonctionner l'onduleur. - Signal IGBT1 : ( +Sinus -Porteuse) - Signal IGBT2 : (Signal IGBT1) - Signal IGBT3 : ( - Sinus +Porteuse) - Signal IGBT4 : (Signal IGBT3) - La cisaille notifie que le câble a été coupé. - Le microcontrôleur envoie aux deux interrupteurs une commande d'arrêt - Post-conditions : Arrêt de la génération des signaux des IGBTs.
