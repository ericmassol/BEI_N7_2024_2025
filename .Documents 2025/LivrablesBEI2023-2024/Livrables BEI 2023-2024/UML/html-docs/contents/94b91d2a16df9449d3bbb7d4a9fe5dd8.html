Description

Cas d'utilisation : UC1M1 Commander interrupteur 1 de la cisaille Acteurs : Module de communication Point de vue : Préconditions : Module de communication actif et en bon fonctionnement, microcontrôleur actif et en bon fonctionnement, ordre donné par l'opérateur. Déroulement Normal : - Description détaillée : Le module de communication communique au microcontrôleur1 l'ordre de fermer l'interrupteur . L'interrupteur se ferme et renvoie à l'acteur une notification de bon fonctionnement. - Postconditions : Notification de bon fonctionnement / interrupteur fermé Déroulement alternatif : Le module de communication communique au microprocesseur l'ordre de fermer l'interrupteur . Défaillance du microprocesseur et l'interrupteur ne se ferme pas. - Exceptions : Microprocesseur HS - Variantes : Module de communication HS, interrupteur HS, batterie HS
